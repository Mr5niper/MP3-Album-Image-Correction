Optional: place a tested LGPL ffmpeg.exe here as bin\ffmpeg.exe and the build
will use it as-is. If this folder has no ffmpeg.exe, BUILD_EXE.bat downloads the
current LGPL build from BtbN automatically and extracts just ffmpeg.exe here.
Use only an LGPL build (not GPL or nonfree) so the release stays
redistributable. See the NOTICE file for details.

(https://github.com/btbn/ffmpeg-builds/releases)
